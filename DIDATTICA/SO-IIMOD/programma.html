Testi consigliati
  • I moderni sistemi operativi, A. Tanenbaum, Prentice Hall International, ed. 1994.
  • Sistemi operativi, A. Silberschats, P. Galvin, Addison Wesley, ed. 1998.
  • LINUX programmer's guide.
  • Programmazione in ambiente UNIX, F. Quaglia, C. Demetrescu, Disp. Didatt., Settembre 1999.
Programma
  • Struttura ed implementazione del sistema UNIX (Tanenbaum cap. 7):
    • Panoramica sul sistema.
    • Processi, modello di memoria, organizzazione del file system, Input/Output.
    • Implementazione di un processo, implementazione della gestione della memoria, implementazione del file system, implementazione dell'Input/Output.

  • Sistemi Distribuiti (Tanenbaum cap. 9.1, 9.4 - Tanenbaum cap. 10 escluso 10.4 - Tanenbaum cap. 11 escluso 11.2 - Tanenbaum cap. 13 - Silberschats Galvin cap. 15 - Silberschats Galvin cap. 17):
    • Introduzione ai sistemi distribuiti: obiettivi di progetto, comparazione con sistemi centralizzati, similitudini con sistemi operativi per macchine parallele.
    • Comunicazione nei sistemi distribuiti:
    • Reti e tipi di comunicazione.
    • Protocolli a livelli: il modello OSI, protocolli Internet (TCP, UDP), comparazione tra protocolli.
    • Il paradigma Client/Server: primitive bloccanti e non, primitive bufferizzate e non, primitive affidabili e non; implementazione del paradigma Client/Server.
    • Servizi remoti: chiamata di procedura.
    • Sincronizzazione: clock logici, clock fisici, algoritmi per la sincronizzazione dei clock.
    • Transazioni atomiche.
    • Stallo nei sistemi distribuiti: tecniche distribuite di scoperta e prevenzione.
    • File system distribuiti: naming, accesso a file remoti, replicazione di file, esempi di file system distribuiti (NFS, AFS).

  • Programmazione in ambiente UNIX (Linux Programmers Guide - Programmazione in ambiente UNIX):
    • Richiami su concetti di base della programmazione in ambiente UNIX: richiami sul linguaggio C, system calls per la gestione dell'I/O, system calls per la gestione dei processi,
    • Comunicazione e sincronizzazione tra processi:
      • Pipes.
      • Named Pipes (FIFO).
      • Code di messaggi.
      • Memoria condivisa.
      • Semafori.
      • Segnali.
      • Sockets.
      • Ping routines.
    • Struttura ed implementazione di un processo server concorrente.

  • Cenni sul sistema Windows NT (Silberschats Galvin cap. 24):
    • Componenti del sistema.
    • File System.
    • Scheduling di processi/thread.
    • Sistema di memoria virtuale.
    • Interfaccia per il programmatore.
